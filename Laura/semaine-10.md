 - Les instructions PHP sont placées dans une balise ouvrante et fermante : <?php ?>
 - Ex: 
 
                      <!DOCTYPE html>
                      <html>
                          <head>
                              <meta charset="utf-8" />
                              <title>Ma page web</title>
                          </head>
                          <body>
                              <h1>Ma page web</h1>
                              <p>Aujourd'hui nous sommes le <?php echo date('d/m/Y h:i:s'); ?>.</p>
                          </body>
                      </html>
